Verification

Two ways to verify an issuance:

  • Human check: open the report, compare VisaID and evidence table consistency.
  • Machine check: validate JSON integrity + declared hashes/signatures (if present in the package).
Machine verification quick steps
1) Download the proof package (JSON) 2) Verify the package self-hash / signature fields (if included) 3) Recompute hashes for referenced artifacts listed in the evidence table 4) Ensure VisaID matches across: header, evidence, and any signed statement

Note: v0.2 supports a standardized structure for third-party verification and future chaining (time anchors / multi-node signatures).
