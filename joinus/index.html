Deploy a node. Emit verifiable presence. Join the observation layer.

Get Started

What This Is

  • A minimal entry into a distributed observation network
  • No image distribution. No complex setup
  • A node is created and begins emitting signals

What You Become

  • A node in a global governance network
  • A verifier of evidence
  • A participant in distributed decision-making

Node Types

  • Core Node — governance + aggregation
  • Edge Node — execution + observation
  • External Node — partner participation

Install

curl -sSL https://89.167.28.102/bootkit/install.sh | bash

Execution will:

  • Generate node identity
  • Attempt registration
  • Start heartbeat automatically
Current release: v1.3
Enrollment is operator-controlled at this stage.

Requirements

  • Debian / Armbian compatible machine
  • Network connection (IPv4/IPv6)
  • “Any machine can become a node”

Node Behavior

  • Each node generates a unique identity
  • Heartbeat establishes observable presence
  • Data is emitted to the network layer
  • Nodes are not deployed. They emerge.

Status

Nodes are visible through the network status layer.
(Public interface under construction)

After Joining

  • Your node receives an identity
  • Participates in governance
  • Produces verifiable certificates
